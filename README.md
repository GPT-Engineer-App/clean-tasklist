# clean-tasklist

A simple and user-friendly to-do app where users can add, edit, and delete tasks. The app should have a clean interface with options to mark tasks as completed and prioritize them. Include due dates for tasks and allow sorting by priority and due date.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/clean-tasklist.git
cd clean-tasklist
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
